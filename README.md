🚧 FixIt

FixIt is a civic engagement app that allows users to report local municipal issues such as potholes, garbage dumps, broken streetlights etc with just a few taps. It uses AI to validate reported issues and redirects them to the appropriate authorities.


✨ Features

- 📍 Automatic location tagging using GPS
- 🧠 AI-powered issue detection and validation
- 🔐 Role-based login system (Admin & User)
- 🛠 Admin dashboard to manage reported issues
- 🔄 Real-time updates with Firebase backend
- 📱 Mobile-responsive and user-friendly UI

🔧 Tech Stack

- Frontend: React + TypeScript + Tailwind CSS 
- Backend: Firebase (Authentication + Firestore)
- AI Integration: Image-based issue classification
- Routing: React Router DOM


🚀 Getting Started

Prerequisites

- Node.js & npm
- Firebase account & project
- Git

Installation
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
